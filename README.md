A premium, high-converting, single-page website designed to showcase and sell fully developed residential and commercial plots in Gwalior, Madhya Pradesh. 

Built with pure HTML, CSS, and Vanilla JavaScript, this project is optimized for speed, mobile responsiveness, and direct client lead generation without the need for a heavy backend.

## ✨ Key Features

* **💎 Premium UI/UX:** Deep navy and gold color scheme engineered to convey trust, stability, and luxury.
* **📱 Mobile-First Design:** Fully responsive layout that looks and functions flawlessly on any smartphone or desktop.
* **🚀 High-Conversion Sticky FABs:** Persistent Floating Action Buttons for WhatsApp and direct phone calls, ensuring the agent is always just one tap away.
* **💬 Pre-filled WhatsApp Integration:** Property inquiry links automatically open WhatsApp with pre-typed messages for specific plot sizes, removing friction for the buyer.
* **🎭 Scroll Reveal Animations:** Smooth, lightweight JavaScript intersection observers that fade elements in as the user scrolls, giving a modern, branded feel.
* **⚡ Zero Dependencies:** No heavy frameworks (like React or Bootstrap). Just fast-loading, clean code.

---

## 🛠️ Usage & Setup

Because this is a static site, there is no complex build process or server setup required.

1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser to view the site locally.
3. To deploy, simply push the code to a GitHub repository and enable **GitHub Pages** from the repository settings.
